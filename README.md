# PILOT REST CALCULATOR

Upload all files in this folder to the root of a public GitHub repository, then enable GitHub Pages from the `main` branch and `/ (root)`.

The published app will use:

- `index.html`
- `pilot-rest-logo-styles.css`
- `pilot-rest-actual-start-app.js`
- `manifest.webmanifest`
- `sw.js`
- `pilot-rest-logo.png`

After GitHub Pages publishes, open the site once while online before testing offline mode.
